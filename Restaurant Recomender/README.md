#  Restaurant Recommender

## Main Idea

- Have interactive map
- Have list of restaurants w/ ratings/ locations using some API
- Have users filter restaurants by location, cuisine, price, open times, if they have already been there
- Give users picks of restaurants, if they have already been there they can mark it down
- Have them able to call/ order/ give directions to restaurant
- After eating ask them to rate and take photos of food as well as rate restaurant as a whole.  
- Use a recommending algorithm to recommend restaurants with popular dishes/ notice what people like in a restaurant -> recommend later. As they build their **eater profile** they will get better and better recommendations. 
- Have restaurants appear on a map w/ features that show if a restaurant is popular / popular dishes.
